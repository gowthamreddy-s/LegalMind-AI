Core Features

Everything a lawyer needs

Built specifically for Indian legal workflows — not a generic chatbot.

💬

Chat with Documents

Ask anything about any uploaded document. Get precise answers with page citations. Answers come strictly from the document — never from external knowledge.

❓

Auto FAQ Generation

Upload a document and get 6 intelligent FAQs generated automatically. Adapts to document type — contract, policy, report, or legal notice.

🔍

Multi-Document Compare

Select 2–5 documents and compare them. Get structured analysis with Executive Summary, Similarities, Differences, Key Findings, and Conclusion.

📂

Private Document Library

Each lawyer's documents are completely isolated. Persistent storage across sessions. Upload once, access forever until you delete.

🔒

Secure Login

Supabase Auth with email/password. Each advocate sees only their own documents. Row-level security ensures complete data privacy.

📊

Confidence Scoring

Every answer shows a retrieval confidence score. Source chunks shown with page references. Full transparency on how answers were generated.

How It Works

The RAG Pipeline

Strict Retrieval-Augmented Generation — answers come only from your documents, never from external knowledge.

1

Upload

PDF / DOCX / CSV / Excel / Image

2

Extract

Text extracted from all file types

3

Chunk

Split into 1000-token overlapping chunks

4

Embed

BGE embeddings stored in ChromaDB

5

Retrieve

Top-8 chunks by semantic similarity

6

Answer

Llama 3.1 70B with strict RAG prompt
